# 操作系统并发编程作业题

## 题目概述

本目录包含两个经典的进程同步问题的解决方案：

### 题目1：阅览室问题
- **场景**：图书馆阅览室有100个座位
- **约束**：满员时读者需等待，有人离开才能进入
- **关键**：资源数量限制

### 题目2：水果盘问题
- **场景**：盘子容量为2，爸爸放苹果，妈妈放橘子，女儿吃苹果，儿子吃橘子
- **约束**：特定生产者-消费者配对
- **关键**：多种产品的同步

## 实现方法对比

| 问题 | PV操作 | 管程 |
|------|--------|------|
| 阅览室 | 1个资源信号量 | 条件变量+计数器 |
| 水果盘 | 3个信号量 | 条件变量+状态管理 |

## 学习要点

1. **阅览室问题**：简单的资源计数问题
2. **水果盘问题**：生产者-消费者的变种，需要区分不同类型的产品
